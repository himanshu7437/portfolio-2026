# Design System

## Theme:

Dark + futuristic storytelling

## Colors:

* Background: #0B0B0F
* Primary: Soft blue/purple gradient
* Text: White / Gray

## Typography:

* Headings: Bold, large
* Body: Clean, readable

## UI Style:

* Glassmorphism (light)
* Smooth transitions
* Minimal but interactive

## Animations:

* Scroll-based reveal
* Parallax
* Micro-interactions

## Inspiration:

* Apple
* Awwwards sites
